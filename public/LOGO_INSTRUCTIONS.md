# Logo Upload Instructions

## Option 1: Upload via Settings Page (Recommended)
1. Log in as Admin or Staff user
2. Navigate to Settings page
3. Select your logo image file
4. Click "Upload Logo"

## Option 2: Manual Upload
1. Place your logo file in the `public` folder
2. Name it one of:
   - `logo.png`
   - `logo.jpg`
   - `logo.jpeg`
   - `logo.gif`
3. The PDF generator will automatically detect and use it

## Logo Specifications
- **Supported formats**: PNG, JPG, JPEG, GIF
- **Maximum size**: 5MB
- **Recommended dimensions**: 200x80 pixels (or similar aspect ratio)
- **Display size in PDF**: 50x20 points (automatically scaled)

## Notes
- The logo appears on the top right of all PDF documents
- If multiple formats exist, PNG is preferred
- The logo is displayed at 50 points width, height is automatically calculated to maintain aspect ratio
